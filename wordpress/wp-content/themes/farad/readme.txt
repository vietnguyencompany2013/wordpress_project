== Description ==

Farad is a clean, creative, feature-rich and powerful multi-purpose theme that is perfect for creative agencies, creative portfolios, freelancers or for any purpose.

== Primary Menu ==

- "Primary Menu" will work best for five to eight menu items at top level.
- "Primary Menu" supports Five level nested menu depth including top level.

== License ==

See the "Help" tab in the "Farad Options" admin page for license, changelog and other informations.

== Installation ==

Manual installation:

1. Upload the `farad` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'farad'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Farad Options for Theme Options
